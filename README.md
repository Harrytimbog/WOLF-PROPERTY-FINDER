## Run Locally   

Clone the project

``` console
git clone git@github.com:Harrytimbog/WOLF-PROPERTY-FINDER.git
```

Go to the project directory

``` console
cd WOLF-PROPERTY-FINDER
```

Install dependencies

``` console
npm install
```

Install gems

``` console
bundle install
```

Start Server

``` console
rails s
```
